Getting Started for React Strapi CMS

How to install strapi CMS in your local
  1. Enter to gigaland-cms directory
  2. Run npm install
  3. After that, run npm run develop to deploy the cms project in your local directory
  4. Open http://localhost:1337/admin to enter the cms dashboard
  5. Change password strapi with new password in your profile setting
    default user: strapi.admin@mail.com , password : Strapisuperadmin1
  6. Enter directory gigaland / gigaland-grey / gigaland-retro
  7. Run npm install
  8. After that, run npm start for work in local your directory at localhost:3000
  9. Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use build directory for upload in your server
Note: Please run npm run develop for gigaland-cms directory & run npm start for front directory gigaland / gigaland-grey / gigaland-retro in same time for work

Create Columns

For a simple two column layout, create a .row and add the appropriate number of .col-md-* columns. As this is a 12-column grid, each .col-md-* spans a number of those 12 columns, and should always add up to 12 for each row (or the number of columns in the parent).

Please read more for Bootstrap grid system here: https://getbootstrap.com/docs/4.5/layout/grid/

<div className="row">
     <div className="col-md-6"> this is 1/2 columns </div>
     <div className="col-md-6"> this is 1/2 columns </div>
</div>

<div className="row">
     <div className="col-md-4"> this is 1/3 columns </div>
     <div className="col-md-8"> this is 2/3 columns </div>
</div>

<div className="row">
     <div className="col-md-3"> this is 1/4 columns </div>
     <div className="col-md-9"> this is 3/4 columns </div>
</div>

Custom background image for div/section

To make custom background image without touching css file, you can use code style="url(your-image-url)" within your div/section tag. See sample below:

style={{backgroundImage: `url(${'./img/background/4.jpg'})`}}

<section id='profile_banner' className='jumbotron breadcumb no-bg' style={{backgroundImage: `url(${'./img/background/4.jpg'})`}}>
    <div className='mainbreadcumb'>
    </div>
  </section>
